# MegaCenter

## Proyecto

Plataforma de orquestación multi-agente AI self-hosted. Arquitectura híbrida basada en LangGraph + n8n + MCP.

## Metodología

**OBLIGATORIO: Usar siempre BMAD-S MCP** para planificación, diseño, implementación y gestión del proyecto.

### Flujo BMAD-S

1. **Análisis** (Fase 1): Brainstorm, Research (Market/Domain/Technical), Create Brief
2. **Planificación** (Fase 2): Create PRD, Create UX
3. **Solucionamiento** (Fase 3): Create Architecture, Create Epics & Stories, Implementation Readiness
4. **Implementación** (Fase 4): Sprint Planning, Create Story, Dev Story, Code Review

### Reglas

- Antes de cualquier tarea de planificación o desarrollo, consultar `bmad_help` para routing
- Seguir el protocolo VRG (Verify/Refine/Generate) en cada paso
- No saltar fases sin completar los artefactos requeridos (marcados como *required*)
- Usar los agentes BMAD-S apropiados para cada workflow (Lisa=PM, Frink=Architect, Homer=Dev, etc.)
- Documentar progreso en `execution-log.yaml`
